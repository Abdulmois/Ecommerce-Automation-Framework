# 🛒 E-Commerce Automation Framework (Selenium + Java)

## 📌 Project Overview

This is an end-to-end automation testing framework built using Selenium WebDriver, Java, and TestNG.
It automates a real-world e-commerce flow including login, product search, add to cart, and validation.

---

## 🚀 Features

* 🔐 User Login Automation
* 🔍 Product Search Functionality
* 🛒 Add Multiple Products to Cart
* 🔄 Multi-Window Handling
* ✅ Cart Count Validation
* ❌ Remove Items from Cart

---

## 🛠 Tech Stack

* Java
* Selenium WebDriver
* TestNG
* Maven
* Page Object Model (POM)

---

## 📂 Framework Design

* `src/main/java` → Page Objects, Base Classes, Utilities
* `src/test/java` → Test Cases
* Follows Page Object Model for maintainability and scalability

---

## ▶️ How to Run

1. Clone the repository
2. Open in Eclipse/IntelliJ
3. Run `testng.xml`

---

## 💡 Key Highlights

* Implemented dynamic waits instead of static waits
* Handled multiple browser windows
* Designed reusable framework structure
* Focused on real-world automation scenarios
